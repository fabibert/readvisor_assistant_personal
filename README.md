Getting started:
- clone repo
- create an `.env`  file in the root dir with content: openai_api_key={your api key}
- `py -m venv .venv`
- `.\.venv\Scripts\activate`
- `py -m pip install -r requirements.txt`
- `flask --app readvisor_ai_assistant run --debug`
